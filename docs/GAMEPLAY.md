# にょわくじ2026 — ゲームデザイン仕様書

## コンセプト

**にょわくじ2026** は「労働拒否」をテーマにしたおみくじアプリです．

アプリ自体が「にょわ〜」と休みたがっており，ユーザーがくじを引こうとすると全力で抵抗します．
最終的に捕まえたときの結果は，どれだけ追いかけたか，どれだけ待ったかによってトーンが変わります．
しつこく追いかければ怒り，ゆっくり待てば機嫌がよくなります．

---

## ゲームループ

1. 待機
2. ユーザーがボタンに触れる
3. 抵抗パターン発動
4. ユーザーが捕まえる
5. おみくじ演出
   - 途中で止まる
   - 逆回転
   - 即出し
   - 通常
6. 結果表示
7. もう一回？
8. 1 に戻る

---

## 抵抗 (回避) パターン

ボタンに hover / touchstart した瞬間にランダムでいずれか 1 つが発動します．

| パターン                | 確率 | 挙動の概要                                                                        |
| ----------------------- | ---- | --------------------------------------------------------------------------------- |
| Dodge (物理回避)        | 30%  | ボタンが画面内のランダム位置に CSS transition で逃げ，追うたびに再逃走            |
| Clone (分身)            | 20%  | ボタンが 3 つに増殖し，ハズレを押すと吹き出しを出して消える                       |
| Excuse (屁理屈)         | 20%  | ボタンが disabled になり吹き出しに言い訳が出て，約 2.5 秒ごとに自動で次の言い訳へ |
| Camo (擬態)             | 15%  | ボタンが背景色に同化して見えなくなり，ランダムな位置に移動する                    |
| Cooperate (素直 / レア) | 15%  | 素直に押せる                                                                      |

### Dodge (物理回避)

- hover (PC) または touchstart (モバイル) でトリガー
- 逃げ回数が増えるほど transition 時間が長くなり「疲れて遅くなる」(`evasion.dodge_transition` 参照)
- 再ホバー / 再タッチするたびに `evade_count +1`

### Clone (分身)

- 3 ボタンが画面内のランダム位置に出現 (うち 1 つだけ本物)
- ハズレを押すと: 吹き出しを出してそのボタンが消える (`evade_count +2`)
- `+2` の理由: Clone は手がかりなしで迷うので Dodge より重く加算してバランス調整

### Excuse (屁理屈)

- ボタンは元の位置のまま disabled (テキストは「くじを引く」のまま固定)
- 吹き出しに言い訳テキストが出る (`content.excuse_text` 参照 / 3 種類)
- 3 回こなし切ると Idle に戻る
  - `evade_count +1` (完走時のみ / 自動進行中は加算しない)
  - `idle_started_at` をリセット (この Idle 戻りが Rested / Sleepy への自然な入口)
  - 吹き出しにヒントを 4 秒表示 (`content.excuse_hint` 参照)

### Camo (擬態)

- ボタンがランダム位置に移動して `background-color`, `color`, `border-color` をすべて背景色にセット
- 位置を当てて押せばキャッチ成功
- `evade_count` は加算なし

### Cooperate (素直)

- 通常ボタンがそのまま押せる
- `evade_count` は加算なし

## 機嫌 (ムード) システム

### evade_count の加算ルール

| タイミング                      | 加算 | 理由                         |
| ------------------------------- | ---- | ---------------------------- |
| Dodge 中に再ホバー / 再タッチ   | +1   | 能動的に追いかけている       |
| Clone のハズレを踏む (1 回ごと) | +2   | 探す手間が重いため重めに加算 |
| Excuse を 3 回こなし切った      | +1   | 完走時のみ                   |
| Camo / Cooperate でキャッチ     | 0    | 抵抗が少ない or なし         |
| パターン発動 (Idle → Evading)   | 0    | ユーザーはまだ何もしていない |

### idle_started_at のリセットタイミング

| タイミング                               | 理由                     |
| ---------------------------------------- | ------------------------ |
| ページ読み込み (init)                    | 最初のラウンド開始       |
| PlayAgain ボタン押下                     | 新しいラウンドの待機開始 |
| Excuse が 3 回完走して Idle に戻った瞬間 | 「休む機会」を与え直す   |

`idle_ms = first_interact_at − idle_started_at` (最初に触れるまでの待ち時間)

### ムード判定優先順

| 優先度 | ムード  | 条件               |
| ------ | ------- | ------------------ |
| 1      | Sleepy  | `idle >= 120s`     |
| 2      | Rested  | `idle >= 30s`      |
| 3      | Furious | `evade_count >= 6` |
| 4      | Grumpy  | `evade_count >= 3` |
| 5      | Neutral | その他             |

#### Rested が Furious より優先される理由

- 「30 秒待った」という辛抱強さは「その後多少追いかけた」事実より重みがあります
- プレイヤーの忍耐を正しく報酬として扱うために idle 条件を先に評価します

### Rested / Sleepy への到達パス

```
通常:      [Idle] → 30秒以上何もしない → 触る → Rested
                  → 2分以上何もしない  → 触る → Sleepy（隠し要素）

Excuse 後: [Excuse×3完走] → Idle に戻る（ヒント表示）
                          → 30秒待つ → 触る → Rested
```

Sleepy は「ブラウザを開いたまま放置する」という完全な隠し要素。Excuse ヒントは Rested への唯一の公式示唆。

## おみくじ演出 (Drawing フェーズ)

キャッチ成功後，結果確定までの演出がランダムに変わります．

| 演出                | 確率 | 結果表示までの時間              |
| ------------------- | ---- | ------------------------------- |
| Instant (即出し)    | 15%  | 900ms                           |
| Spinning (通常回転) | 40%  | 2800ms                          |
| Paused (途中停止)   | 25%  | 1200ms 後に停止 / 3500ms で結果 |
| Reversing (逆回転)  | 20%  | 1200ms 後に逆転 / 3400ms で結果 |

吹き出しと顔文字の対応は `content.gleam` (`draw_*`) と `view.gleam` (`drawing_view`) を参照してください．

## コンテンツの所在

すべてのユーザー向けテキスト / 顔文字 / おみくじ結果は `src/nyowa/content.gleam` に集約されています．

テキストを変えたいときはこのファイルだけ編集すればよいです．

| 変更したいもの               | `content.gleam` の定数 / 関数                                                                                      |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| アプリタイトル・サブタイトル | `app_title`, `app_subtitle`                                                                                        |
| ボタンラベル                 | `draw_button_label`, `play_again_label`                                                                            |
| Idle 時のデフォルト吹き出し  | `idle_default`                                                                                                     |
| 各パターンの最初の吹き出し   | `dodge_initial`, `clone_initial`, `camo_initial`, `cooperate_initial`                                              |
| Dodge 追いかけ時の吹き出し   | `dodge_dialogue(count)`                                                                                            |
| Clone ハズレ時の吹き出し     | `clone_fake_hit`                                                                                                   |
| Excuse の言い訳テキスト      | `excuse_text(index)`                                                                                               |
| Excuse 完走後のヒント        | `excuse_hint`                                                                                                      |
| Drawing 各状態の吹き出し     | `draw_instant`, `draw_spinning`, `draw_paused`, `draw_reversing`                                                   |
| スロットドラムに流れる文字列 | `drum_names`                                                                                                       |
| おみくじ結果 (ムードごと)    | `furious_fortunes`, `grumpy_fortunes`, `neutral_fortunes`, `rested_fortunes`, `sleepy_fortunes`                    |
| キャラクター顔文字           | `char_calm`, `char_tense`, `char_happy`, `char_focused`, `char_upset`, `char_flat`, `char_cheerful`, `char_drowsy` |
